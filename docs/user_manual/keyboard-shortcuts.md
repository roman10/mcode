# Keyboard Shortcuts

Press `Cmd+/` at any time to open the shortcut reference dialog inside mcode.

## General

| Shortcut | Action |
|---|---|
| `Cmd+N` | New Claude session |
| `Cmd+T` | New terminal |
| `Cmd+Shift+T` | New task |
| `Cmd+P` | Quick Open (file search) |
| `Cmd+Shift+P` | Command Palette |
| `Cmd+\` | Toggle sidebar |
| `Cmd+/` | Show keyboard shortcuts |
| `Cmd+,` | Open settings |
| `Cmd+Shift+A` | Show Activity |
| `Cmd+Shift+B` | Show Stats |
| `Cmd+Shift+C` | Show Changes |
| `Cmd+Shift+E` | Run Shell Command |
| `Cmd+Shift+F` | Search in Files |
| `Cmd+Shift+L` | Toggle layout mode |
| `Cmd+Shift+M` | Clear all attention |
| `Cmd+Shift+S` | Snippets |
| `Cmd+R` | Refresh Stats |
| `Ctrl+\`` | Toggle Terminal Panel |
| `Cmd+W` | Close tile |
| `Cmd+Shift+X` | Close all tiles |

## Sessions

| Shortcut | Action |
|---|---|
| `Cmd+1` – `Cmd+9` | Focus session by position |
| `Cmd+]` | Next session |
| `Cmd+[` | Previous session |
| `Cmd+F` | Filter sessions |
| `F2` | Rename session |
| `Enter` | Open session tile |
| `⌫` | Kill / delete session |

## Terminal

| Shortcut | Action |
|---|---|
| `Cmd+C` | Copy selection |
| `Cmd+A` | Select all |
| `Cmd+K` | Clear terminal |
| `Cmd+F` | Find in terminal |
| `Cmd+Shift+W` | Kill session & close tile |
| `Cmd+D` | Split horizontal |
| `Cmd+Shift+D` | Split vertical |
| `Cmd+Enter` | Toggle maximize |
| `Cmd+=` | Zoom in |
| `Cmd+-` | Zoom out |
| `Cmd+0` | Reset zoom |

## Search (when search bar is open)

| Shortcut | Action |
|---|---|
| `Enter` | Next result |
| `Shift+Enter` | Previous result |
| `Cmd+G` | Next result |
| `Cmd+Shift+G` | Previous result |
| `Escape` | Close search |
