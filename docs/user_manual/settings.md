# Settings

Open settings with `Cmd+,` or click the gear icon in the sidebar footer.

## General

| Setting | Default | Description |
|---|---|---|
| Prevent sleep while sessions are active | On | Keeps your Mac awake while Claude sessions are running. Screen dimming is unaffected. |

## Commit Tracking

| Setting | Default | Description |
|---|---|---|
| Scan all branches | Off | When off, only commits on the main branch are tracked. Turn on to include commits from all branches. |
